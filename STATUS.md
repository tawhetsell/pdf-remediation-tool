# Remediation Status

Per-file state. Update after every pipeline run and every Canvas upload.

## Columns

- **Canvas name**: filename in `work/canvas_ready/<module>/`
- **Module**: folder in `canvas_ready/`
- **BDC**: marked-content operator count from local verify
- **Ally score**: `Perfect`, `High`, `Medium`, `Low`, `—`
- **Date**: YYYY-MM-DD of Canvas verification
- **Notes**: non-standard outcome or follow-up needed
